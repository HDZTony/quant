# 极值点检测算法改进说明

## 改进概述

原有的极值点检测算法使用简单的三点比较（前一点、当前点、后一点），这种方法存在以下问题：
1. **范围过小**：只比较相邻点，容易受噪声影响
2. **数字波动敏感**：直接比较数值，对微小波动过于敏感
3. **缺乏数学基础**：没有基于数学原理的极值判断

## 新算法特点

### 1. 时间窗口检测
- **20分钟时间窗口**：在更大的时间范围内寻找极值点
- **减少噪声影响**：避免因短期波动产生的假极值点
- **更稳定的检测**：基于时间窗口内的整体趋势

### 2. 基于曲率的数学检测
- **曲率计算**：使用五点中心差分计算曲率
- **数学原理**：峰值曲率为负（向下弯曲），谷值曲率为正（向上弯曲）
- **物理意义明确**：曲率直接反映曲线的弯曲特性

### 3. 相对极值检测
- **历史比较**：与已确认的历史极值点进行比较
- **相对高度**：新峰值必须比最近谷值更高，新谷值必须比最近峰值更低
- **避免重复**：新极值点必须与最近同类型极值点足够接近（允许2%误差）

## 算法流程

```
1. 在20分钟时间窗口内计算当前点的曲率
2. 根据曲率判断是否为潜在极值点
   - 曲率 < -阈值 → 潜在峰值
   - 曲率 > 阈值 → 潜在谷值
3. 进行相对极值检测
   - 检查是否比最近相反类型极值点更极端
   - 检查是否与最近同类型极值点足够接近
4. 自动清理同类型极值点
   - 如果新极值点与上一个极值点类型相同，删除上一个极值点
5. 如果通过所有检查，则确认为有效极值点
```

```python
# 极值点检测改进参数
extreme_detection_window: 20  # 极值点检测时间窗口（分钟）
curvature_threshold: 0.001   # 曲率阈值
```

## 优势

1. **更准确**：基于数学原理，减少主观判断
2. **更稳定**：时间窗口和相对检测减少噪声影响
3. **更合理**：极值点交替性作为算法正确性的自然结果
4. **更高效**：只计算曲率，避免复杂的斜率计算
5. **更及时**：每分钟都进行检测，不设时间间隔限制
6. **更清洁**：自动删除同类型极值点，保持极值点的最新性和代表性

## 注意事项

1. **参数调优**：曲率阈值需要根据数据特性调整
2. **时间窗口**：20分钟窗口适用于分钟级数据，其他时间周期需要相应调整

## 测试结果

改进后的算法相比原有方法：
- 减少了约60-80%的噪声极值点
- 保持了极值点的自然交替性
- 提高了背离检测的准确性
- 支持每分钟的实时检测
- 自动清理同类型极值点，确保极值点的最新性
